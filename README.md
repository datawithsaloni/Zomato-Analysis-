# 🍽️ Zomato Analytics Dashboard

## 📌 Project Overview
The Zomato Analytics project focuses on analyzing restaurant data to understand customer preferences, restaurant performance, ratings, sales trends, and delivery services. Using Excel, Power BI, Tableau, and SQL, this project provides interactive dashboards and business insights to support restaurant and food delivery analytics.

The project demonstrates end-to-end analytics including data cleaning, KPI analysis, SQL querying, and dashboard development.

---

## 🎯 Project Objectives
- Analyze restaurant performance and customer ratings
- Identify top reviewed restaurants and popular cuisines
- Study restaurant opening trends by year and month
- Evaluate online delivery and table booking availability
- Generate business insights using SQL queries
- Build interactive dashboards for data-driven decision-making

---

## 🛠️ Tools & Technologies Used
- **Microsoft Excel** – Data Cleaning, Pivot Analysis & Dashboard
- **Power BI** – Interactive Dashboard & Data Visualization
- **Tableau** – Visual Analytics & Storytelling Dashboard
- **SQL / MySQL** – Data Querying and KPI Analysis

## 📊 Key Performance Indicators (KPIs)

This project analyzes the following KPIs:

- Total Number of Restaurants – 1,284
- Average Rating – 4.3
- Total Sales – $2M
- Most Reviewed Restaurant – Barbeque Nation
- Top 5 Most Reviewed Restaurants
- Restaurants Opened by Year
- Restaurants Opened by Month
- Top 5 Cuisines
- Online Delivery Percentage
- Table Booking Availability

---

## 💡 Key Insights
Some major insights derived from the analysis include:

- Barbeque Nation received the highest number of customer reviews
- North Indian and Chinese cuisines were the most popular
- Restaurant openings peaked during selected years and months
- Online delivery availability influenced restaurant reach
- Table booking options were limited across restaurants
- Ratings and customer engagement helped identify top-performing restaurants
